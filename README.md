## [리액트 모듈 프로젝트] 문화 컨텐츠 사이트 설계 및 구현

[기관/과정명]
- 한국품질재단 / 부산 클라우드 네이티브 애플리케이션 개발자 양성과정

[사용 기술]
- HTML / CSS / React JSX

[사용 툴]
- VS Code

[맡은 역할]
- 뮤지컬 디테일 페이지와 예매 기능 담당
- Axios와 Postman을 이용해서 데이터 가져오기
- 최종 PPT 발표

[구현 결과]
- 사용자 친화적인 UI를 생각해서 구현
- 예매 기능에서 예외 처리를 완벽하게 하지 못한 것이 아쉬움
